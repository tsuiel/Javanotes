Java Thread子类创建线程 笔记

1. 概念：线程是程序执行的最小单位，继承 Thread 类是创建线程的一种方式，自定义类继承Thread后，重写run()方法定义线程任务。

2. 特点：调用 start() 方法启动线程（而非直接调用run()），线程执行顺序由CPU调度决定，具有随机性。

3. 常用方法：start() 启动线程；run() 线程执行的任务；Thread.sleep(ms) 让线程休眠指定毫秒。

4. 注意：直接调用run()只是普通方法调用，不会开启新线程；一个线程对象只能调用一次start()，重复调用会抛出异常。

Thread子类创建线程 示例代码
public class ThreadDemo {
    public static void main(String[] args) {
        //创建自定义线程对象
        MyThread t1 = new MyThread("线程1");
        MyThread t2 = new MyThread("线程2");

        //调用start()方法启动线程
        t1.start();
        t2.start();
    }
}

//定义Thread子类
class MyThread extends Thread {
    private String name;

    public MyThread(String name) {
        this.name = name;
    }

     //重写run()方法，定义线程任务
    @Override
    public void run() {
        for (int i = 0; i < 5; i++) {
            System.out.println(name + " 执行：" + i);
            try {
                Thread.sleep(100); //线程休眠100毫秒
            } catch (InterruptedException e) {
                e.printStackTrace();
            }
        }
    }
}
