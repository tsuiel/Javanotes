Java 类与对象 基础笔记

1. 类的概念：类是对象的模板，是对一类事物的共同属性和行为的抽象。

2. 对象的概念：对象是类的实例，是类的具体实现，通过 new 关键字创建。

3. 类的组成：成员变量（属性）、成员方法（行为）、构造方法。

4. 构造方法：方法名与类名完全一致，没有返回值，用于对象初始化。

5. 访问修饰符：public、private、protected、默认，控制类和成员的访问权限。

类与对象 完整示例代码

// 定义一个学生类
public class Student {
    // 成员变量（属性）
    private String name;
    private int age;

    // 构造方法
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }

    // 成员方法（行为）
    public void showInfo() {
        System.out.println("姓名：" + name + "，年龄：" + age);
    }

    // 主方法，程序入口
    public static void main(String[] args) {
        // 创建对象
        Student s = new Student("张三", 18);
        // 调用方法
        s.showInfo();
    }
}

核心知识点补充

this关键字
作用1：区分成员变量和局部变量
作用2：在构造方法中调用本类其他构造方法

封装
使用private修饰成员变量，提供get/set方法访问，保护数据安全

匿名对象
new Student("李四", 20).showInfo();
特点：只能使用一次，没有引用变量指向
