Java 泛型 基础笔记

1. 概念：泛型是参数化类型，让类、接口或方法支持多种数据类型，提高代码复用性与类型安全性。

2. 特点：编译期类型检查、避免强制类型转换、不支持基本数据类型（需使用包装类）。

3. 分类：泛型类、泛型接口、泛型方法、通配符泛型。

4. 常用符号：<T> 代表类型参数，<?> 代表无界通配符。

5. 注意：泛型类型信息在编译后会被擦除（类型擦除），运行时不保留泛型信息。

泛型示例代码
public class GenericDemo {
  public static void main(String[] args) {
    // 泛型类使用
    GenericBox<String> strBox = new GenericBox<>("Hello");
    System.out.println(strBox.getData());

    // 泛型方法调用
    Integer num = GenericMethod.showType(100);
    System.out.println(num);
  }
}

// 泛型类
class GenericBox<T> {
  private T data;
  public GenericBox(T data) {
    this.data = data;
  }
  public T getData() {
    return data;
  }
}

// 泛型方法
class GenericMethod {
  public static <T> T showType(T t) {
    return t;
  }
}
